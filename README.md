# 智能垃圾分类系统：从粗放到精细的技术演进

## 项目概述
本项目旨在通过深度学习技术，实现垃圾图像的精准分类。经历了从 **四大类原型验证** 到 **四十类细粒度攻坚** 的完整演进过程。

## 技术演进路径
1. **V1：原型验证阶段**（已完成）
   - 目标：验证技术可行性
   - 方法：四大类（可回收、厨余、有害、其他）快速分类
   - 结论：可行，但无法指导具体投放，需细化。

2. **V2：细粒度攻坚阶段**（本阶段）
   - 目标：实现精准可操作的分类
   - 挑战：40类样本不平衡、类间相似度高、开放世界问题
   - 核心解决方案：
     - 数据层面：针对性的增强与重采样
     - 模型层面：YOLO分类模型优化与置信度过滤
     - 系统层面：设计未知物处理机制

## 📌 项目概述
本项目旨在构建一个精准的细粒度垃圾分类模型，能够区分40个具体的垃圾类别（如“塑料瓶”、“果皮”、“电池”等），而不仅仅是四大类。项目完整涵盖了从**原始数据处理、模型训练优化到Web应用部署**的全流程。

**核心演进**：从最初的“四大类”原型验证，升级至当前解决实际分类需求的“40类细粒度”攻坚。

## 🧹 数据清洗（核心工作）
在数据准备阶段，我们对原始数据集进行了关键的人工清洗与整理，这为模型性能奠定了坚实基础。
- **问题发现**：
    1.  **类别合并**：原数据集中“厨余垃圾剩饭剩菜”与“厨余垃圾大骨头”两个类别被错误合并于同一文件夹。
    2.  **标签噪声**：部分类别中存在错位图片（如在“药膏”文件夹中发现“酒瓶”图片）。
- **解决方案与行动**：
    1.  **手动拆分**：将合并文件夹中的图片，依据视觉特征人工复核并拆分至两个独立的正确类别文件夹。
    2.  **全面检查与纠正**：对所有40个类别进行抽样检查，将发现的明显错误图片移动至其对应的正确类别下。
- **清洗后数据统计**：
    - 总图片数：[37300张]
    - 总类别数：40
    - 图片最多的类别：[可回收物插头电线] (817张)
    - 图片最少的类别：[其他垃圾牙签] (150张)
    - *（数据分布较为健康，为后续处理提供了良好基础）*
