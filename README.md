# 智能垃圾分类系统：从粗放到精细的技术演进

## 项目概述
本项目旨在通过深度学习技术，实现垃圾图像的精准分类。经历了从 **四大类原型验证** 到 **四十类细粒度攻坚** 的完整演进过程。

## 技术演进路径
1. **V1：原型验证阶段**（已完成）
   - 目标：验证技术可行性
   - 方法：四大类（可回收、厨余、有害、其他）快速分类
   - 结论：可行，但无法指导具体投放，需细化。

2. **V2：细粒度攻坚阶段**（本阶段）
   - 目标：实现精准可操作的分类
   - 挑战：40类样本不平衡、类间相似度高、开放世界问题
   - 核心解决方案：
     - 数据层面：针对性的增强与重采样
     - 模型层面：YOLO分类模型优化与置信度过滤
     - 系统层面：设计未知物处理机制

3. **V3：未来构想**（技术展望）
   - 分层分类系统架构
   - 用户反馈驱动的持续学习闭环
